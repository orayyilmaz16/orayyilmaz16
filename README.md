## Hakkımda
Web tabanlı ürünlerde uçtan uca geliştirme yapıyorum: **UI**, **API** ve **veri erişimi** katmanlarında sürdürülebilir, okunabilir ve ölçeklenebilir çözümler üretmeye odaklanıyorum.  
Özellikle **.NET (C#)** ve **modern JavaScript ekosistemi (React)** ile proje geliştirmeyi seviyorum.

### Odak Alanları
- Katmanlı mimari, modüler tasarım, bakım maliyeti düşük kod
- Performans ve kullanıcı deneyimi (UX) odaklı arayüzler
- RESTful servis tasarımı, güvenli kimlik doğrulama/yetkilendirme yaklaşımı
